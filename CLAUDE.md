# 專案環境資訊

這個專案沒有任何環境設定，所以執行後請不要在本地進行任何測試，都是無效的。

## 重要注意事項
- 不要嘗試在本地運行或測試應用程式
- 所有測試和執行都應該在實際部署環境中進行
- 本地執行不會產生有效的結果

## Git 操作說明

docs\prompts.md中的項目完成執行標準git操作：`git add` → `git commit` → `git push`
- commit message 使用英文，不包含協作資訊
- 推送至遠端 master 分支
